Direct Path Load - Como configurar

by Gilberto C. Andrade on 01 junho 2010

Tagged as: Database, Linux, oracle,

Verificando posts de [meu antigo blog][1] notei que alguns não foram importados ou foram importados com defeitos em seu conteúdo. Então resolvi realizar um trabalho de revisão e o [terceiro][2] começa hoje:

Para que se possa usar a opção de [carga direta usando o SQL Load][3], você precisa primeiro configurar o banco de dados para tal atividade. A configuração é simples, basta você executar o script catldr.sql, localizado na instalação do produto, $ORACLE_HOME/rdbms/admin. Depois é só utilizar a opção direct=true na linha de comando.
