Para resolver de vez os problemas com caracteres estranhos e acentuação:



zypper in bundle-lang-common-pt bundle-lang-gnome-pt bundle-lang-kde-pt

Depois, yast->system->language, ajuste o item "Primary Language" para Português(Brasil).
Configure também como as variáveis "locale" serão ajustadas para o usuário root através do botão "Details".
Estes ajustes são gravados no arquivo de configuração /etc/sysconfig/language.
