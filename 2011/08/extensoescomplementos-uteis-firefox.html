Extensões (complementos) FireFox

by Gilberto C. Andrade on 11 agosto 2011

Tagged as: FireFox, Internet,

As seguintes [extensões (complementos)][1] para o [navegador FireFox][2] melhoram bastante a navegabilidade:

Complemento Utilidade
download-status-bar Gerenciador de Downloads
DownThemAll! Gerenciador de Downloads
FlashGot Mass Downloader Gerenciador de Downloads
FireFTP Cliente FTP
Super Start Bookmarks Visuais
GMarks Sincronizar e gerenciar seus Favoritos com o Google
Bookmarks
Dicionário pt-BR Dicionário português brasileiro para o verificador ortográfico integrado do Firefox
Easy Youtube Video Downloader Express Utilitário para baixar vídeos
Ant Video Downloader com Player FLV incorporado
video-downloadhelper Utilitário para baixar vídeos
html5-video-everywhere Utilitário para baixar vídeos
Print Edit Permite editar o conteúdo da página antes de imprimir
Capture & Print Capture & Print permite capturar uma área da página e salvá-la em disco ou enviar para impressora
Session Manager Session Manager saves and restores the state of all or some windows - either when you want it or automatically at startup and after crashes. It can also automatically save the state of open windows individually.

Existem mais complementos, alias uma infinidade deles, que ajudam e transformam o [navegador FireFox][2] em um aplicativo de uso diário. Estes são prioridade para mim e com este post posso buscar e instalar rapidamente.
